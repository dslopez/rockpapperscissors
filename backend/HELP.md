# Running the REST API

Run the api embebed in Tomcat 9:
mvn spring-boot:run

For debuging:
mvn spring-boot:run -Dspring-boot.run.jvmArguments="-Xdebug -Xrunjdwp:transport=dt_socket,server=y,suspend=y,address=5005"
